# 💻 My Calculator

A simple and responsive calculator built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features
- Basic arithmetic: +, −, ×, ÷
- Percent (%), power (xʸ), decimal (.)
- Clear (AC) and backspace (←) functionality
- Fully responsive design

## 🎨 Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
